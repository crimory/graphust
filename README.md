# graphust

Produce simple graphs from basic text input

## Example

### Input
```
A -> B
B -> C
C -> A
```

### Output
```
+---+                   +---+
| A |<------------------| C |
+---+                   +---+
  |                       ^
  |                       |
  |         +---+         |
  --------->| B |---------|
            +---+
```
